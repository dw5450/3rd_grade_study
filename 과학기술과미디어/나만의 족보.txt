


기호 
	물질적인거 아님, 연관성 x(자의적)

	세상은 기호로 이해되고 나타내짐





의미 작용

	기표 : 쓰기 (여러가지로 해석)

	의미작용 : 선택 (여러가지 해석 중 하나)
	
	기의 : 읽기 (해석된 하나)

	퍼스 구분

		도상 : 대상과 비슷 ex) 사진
			
		지표 : 인과 관계 	연기 -> 불

		상징 : 임의로 만들어짐  숫자




코드화
	기의 기표 간 약속
	
	사회적 약속, 관습화




소통 
	기표를 전달, 기의 공유 //나만의 해석

	단순 모델		//나만의 해석
		
		송신자 : 코드화 된 메세지 전송

		채널 : 전달 매개체

		수신자 : 코드화된 매세지 해석

	상호작용성과 참여 인구수로 구분 가능





미디어 
	소통의 시공간 확장 매개체

	지각 환경 바꿈
		
		ex) 책 > 집중해서 책만 , 차 > 거리감이 줄어듬, 영화 > 여러 감각이 동시에

	감각 균형 영향
		
		구술 미디어 : 짧은 거리 상호작용, 복합적

		문자 미디어 : 파편적, 테스트 안의 세계

		전기 미디어 : 최대한의 감각 확장, 즉각성,

		>> 차가운 미디어(감각 > 정보) > 뜨거운 미디어  (감각 < 정보) > 차가운 미디어

	세상을 지각,이해,의식하는 방식 변경

		구술 미디어 : 공동체 의식, 비슷한 지식 공유

		문자 미디어 : 평면적 사고, 인과관계, 분류

		인쇄 미디어 : 파편화 + 사회적 구심화

		전기 미디어 : 감성적, 통합적

		
	세상을 바꿈
		
		구술 미디어 : 마을 단위의 생활

		문자 미디어 : 국가, 계급 생성

		인쇄 미디어 : 문자 문화 증가, 지식인들의 증가, 문자의 통합, 혁명

		전기 미디어 : 지구촌, 부족 사회와 닮음
