
기호

	세상 모든 것은 기호로 나타내짐

	기호는 물질적인 것이 아님 (실체를 잠시 소거)

	기호를 통해 타자와 세상을 이해

	'나'라는 자아와 '의식' 기호로 의루어짐




의미 작용
		
	기호에 의미가 부여하는 작용

	없으면 세계는 알 수 없고 나는 혼돈 상태 이다
		
	기표 + 기의
		
		기표 : 쓰기 (여러가지로 해석 될 수 있음)

		의미작용 : 선택 (여러가지 해석 중 하나를 선택)

		기의 : 읽기 (해석된 하나)

		자의적임 (연관성 x)

	퍼스 구분

		도상 : 대상과 비슷 ex) 사진
			
		지표 : 인과 관계 	연기 -> 불

		상징 : 임의로 만들어짐  숫자



	코드화
		기의와 기표간의 약속된 관계
		
		사회적인 약속, 관습화 되어짐



소통
	기표를 전달하고 기의를 공유하는 과정 //자의적 해석

	미디어

		소통의 시공간을 확장해주는 매개체


	단순 모델 //자의적인 해석, 정확한건 ppt 참조

		송신자 

			코드화 된 메세지 전송


		채널 (미디어, 공기)

			기표가 전달 되는 매개체


		수신자

			코드화 된 매세지 해석, 피드백
